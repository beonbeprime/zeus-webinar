# Task 18 - Escrever Copy dos Anuncios

> Agente responsavel: Clique (ad-copy-specialist)
> Fase: 3 - Copy
> Bloqueante: NAO

## Inputs

- `promise-{project}.md` - Promessa aprovada
- `headlines-{project}.md` - Headlines e hooks aprovados
- `icp-{project}.md` - Perfil do cliente ideal

## Protocolo de Execucao

### STEP 1 - Escrever copy por angulo de dor
Criar 3 variacoes de copy focadas na dor principal do ICP. Estrutura: hook de dor (2 linhas) + agitacao do problema (2-3 linhas) + promessa de solucao (1-2 linhas) + CTA. Tom empatetico, nao dramatico demais.

### STEP 2 - Escrever copy por angulo de oportunidade
Criar 3 variacoes focadas na oportunidade. Estrutura: hook de oportunidade (2 linhas) + o que muda quando resolve (2-3 linhas) + como o webinario entrega isso (1-2 linhas) + CTA. Tom aspiracional, concreto.

### STEP 3 - Escrever copy por angulo de curiosidade
Criar 3 variacoes focadas em curiosidade. Estrutura: hook de curiosidade (2 linhas) + loop aberto sobre o que sera revelado (2-3 linhas) + por que importa (1 linha) + CTA. Tom misterioso, nao clickbait.

### STEP 4 - Escrever copy por angulo de prova
Criar 3 variacoes focadas em prova social. Estrutura: hook com resultado/numero (2 linhas) + contexto do resultado (2-3 linhas) + convite para aprender como (1-2 linhas) + CTA. Tom factual, confiante.

### STEP 5 - Escrever copy por angulo de contraste
Criar 3 variacoes focadas em antes/depois. Estrutura: hook com contraste (2 linhas) + descricao do "antes" (2 linhas) + descricao do "depois" (2 linhas) + como chegar la (1 linha) + CTA. Tom transformacional.

### STEP 6 - Escrever copy de remarketing
Criar 3 variacoes para remarketing (quem clicou mas nao se inscreveu). Tom: "vi que voce se interessou", "a vaga ainda esta aberta", "X pessoas ja se inscreveram". Mais curto, mais direto, mais urgente.

### STEP 7 - Escrever copy de replay
Criar 2 variacoes para anuncios de replay (pos-webinario). Tom: "mais de X pessoas assistiram", "se voce perdeu, ainda da tempo", destaque de 1 insight revelado. CTA para replay com deadline.

### STEP 8 - Submeter para aprovacao
Apresentar toda a copy para aprovacao de Lupa (revisao) e Corte (quality gate).

## Outputs

- `ad-copy-{project}.md` - Copy completa de anuncios por angulo, remarketing e replay, com variacoes para teste

## Checklist de Validacao

- [ ] 3 variacoes por angulo (dor, oportunidade, curiosidade, prova, contraste)
- [ ] 3 variacoes de remarketing
- [ ] 2 variacoes de replay
- [ ] Hooks fortes nas 2 primeiras linhas de cada copy
- [ ] CTAs claros e diretos em todos os anuncios
- [ ] Tom adequado para cada angulo
- [ ] Acentuacao perfeita em todo texto
- [ ] Copy aprovada por Lupa e Corte
