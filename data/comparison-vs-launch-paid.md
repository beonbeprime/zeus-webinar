# Comparacao: Webinario Infalivel vs Lancamento Pago

Tabela de decisao para responder "e melhor fazer webinario ou lancamento pago?"

## Tabela de Decisao

| Criterio | Favorece Webinario (Tayoba) | Favorece Lancamento Pago (Baldan) |
|----------|---------------------------|----------------------------------|
| Ticket do produto | Qualquer (R$97 a R$20k+) | Medio-alto (R$997+) |
| Escala desejada | Alta (pode automatizar/evergreen) | Media (logistica presencial/online) |
| Equipe disponivel | Pequena (1-3 pessoas) | Media-grande (9+ funcoes) |
| Orcamento de trafego | Menor (CPL R$5-10) | Maior (CPA ingresso R$70+) |
| Frequencia | Semanal/mensal (evergreen possivel) | Trimestral/semestral (evento unico) |
| Automacao | Alta (modelo evergreen) | Baixa (evento ao vivo) |
| Comparecimento benchmark | 30%+ (mais dificil reter online) | 50-70% (ingresso pago filtra) |
| Duracao do evento | 60-120 minutos | 2 dias completos |
| Pitch | 1 momento (~1h no webinario) | 3 momentos distribuidos em 2 dias |
| Modelo de negocio | Infoproduto, consultoria, SaaS | Mentoria, imersao, programa presencial |
| Construcao de autoridade | Alta (webinario constroi ao longo) | Media (evento valida autoridade existente) |
| Replay | Parcela substancial (30-50% faturamento) | Gravacao VIP + Sala VIP pos-evento |
| Complexidade operacional | Moderada (stack digital) | Alta (equipe, logistica, contingencia) |
| Validacao rapida | Alta (pode testar em 2-3 semanas) | Baixa (planejamento 30+ dias) |
| Recorrencia | Facil (evergreen, semanal, mensal) | Dificil (evento grande, semestral) |
| ROAS referencia | 3-4x+ | 5x+ |

## Quando usar WEBINARIO

- Produto digital com ticket R$97 a R$5.000
- Equipe pequena (1-3 pessoas)
- Quer escalar com automacao (evergreen)
- Precisa construir autoridade durante o evento
- Quer testar e iterar rapidamente
- Modelo SaaS, consultoria, infoproduto
- Orcamento de trafego limitado
- Quer frequencia alta (semanal/mensal)

## Quando usar LANCAMENTO PAGO

- Mentoria ou programa com ticket R$997+
- Equipe media-grande disponivel (9+ funcoes)
- Autoridade ja existente (evento valida)
- Quer evento de alto impacto (2 dias)
- Produto que exige experiencia presencial/imersiva
- Orcamento de trafego robusto
- Frequencia trimestral/semestral
- Ingresso pago como filtro de qualidade

## Quando COMBINAR

- Webinario como porta de entrada (R$97-497) > upsell para evento presencial (R$997+)
- Webinario evergreen para captacao continua + lancamento pago trimestral para high ticket
- Webinario para validar promessa/oferta antes de investir em lancamento pago
- Lancamento pago para clientes existentes + webinario para novos leads

|---------|-------------------|-------------------------|
| CPL referencia | < R$10 | CPA ingresso R$70 |
| Conversao captura | 25%+ | N/A (ingresso pago) |
| Comparecimento | 30%+ | 50-70% |
| Conversao | 3%+ dos presentes | 15-20% (R$997) / 10-13% (R$1.997) |
| ROAS | 3-4x+ | 5x+ |
| Monetizacao/1000 | Variavel | R$300k-400k/1000 ingressos |
| Replay/pos-evento | 30-50% do faturamento | +65-100% do D1 |
| Faturamento referencia | Variavel por modelo | R$740k-2.1M (cases Baldan) |

## Decisao Rapida

Responda estas 3 perguntas:

1. Seu ticket e abaixo de R$997? > WEBINARIO
2. Voce tem equipe de 9+ pessoas? > LANCAMENTO PAGO
3. Quer escalar com automacao? > WEBINARIO

Se 2 ou 3 apontam para o mesmo: siga essa direcao.
Se empatou: comece com WEBINARIO (mais rapido de validar), evolua para LANCAMENTO PAGO quando tiver dados.
