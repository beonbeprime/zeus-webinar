# Conceitos Visuais de Referência para Eventos

## Os 8 Arquétipos e suas Linguagens Visuais

### Arquétipo Herói
Energia: conquista, superação, ação
Cores dominantes: azul profundo, vermelho vibrante, branco puro
Tipografia: sem serifa bold (Montserrat, Inter Black, Barlow Condensed)
Elementos visuais: setas, movimento, contraste alto
Eventos típicos: negócios, performance, esporte, resultados financeiros

### Arquétipo Sábio
Energia: autoridade, conhecimento, clareza
Cores dominantes: azul marinho, verde escuro, chumbo, bege intelectual
Tipografia: serifa clássica (Playfair Display, Libre Baskerville, Cormorant)
Elementos visuais: espaço em branco, tipografia protagonista, grid limpo
Eventos típicos: educação premium, consultoria, medicina, direito

### Arquétipo Mago
Energia: transformação, mistério, poder
Cores dominantes: roxo intenso, dourado, preto, azul-petróleo
Tipografia: serifa com personalidade + sans para contraste
Elementos visuais: gradientes profundos, brilho, partículas, luz
Eventos típicos: espiritualidade, autoconhecimento, desenvolvimento humano

### Arquétipo Criador
Energia: inovação, autenticidade, possibilidade
Cores dominantes: paletas incomuns, pastéis contrastados, tons terrosos
Tipografia: mista (serifa + sans, tamanhos variados), experimental
Elementos visuais: formas orgânicas, texturas, composição quebrada
Eventos típicos: design, arte, criatividade, branding, moda

### Arquétipo Cuidador
Energia: acolhimento, conexão, segurança
Cores dominantes: verde suave, laranja quente, bege, off-white
Tipografia: sans arredondada e humanista (Nunito, Poppins, Lato)
Elementos visuais: curvas, calor, proximidade, fotografia real
Eventos típicos: saúde, maternidade, relacionamentos, bem-estar, pet

### Arquétipo Fora-da-lei
Energia: ruptura, provocação, autenticidade radical
Cores dominantes: preto, vermelho, amarelo neon, paletas incomuns
Tipografia: impactante, não convencional, às vezes irregular
Elementos visuais: disruptivo, anticonvencional, textura crua
Eventos típicos: disrupção de mercado, mindset, nicho jovem, tech

### Arquétipo Explorador
Energia: aventura, liberdade, descoberta
Cores dominantes: verde floresta, terra queimada, céu azul, areia
Tipografia: sans moderna com personalidade (Raleway, Josefin Sans)
Elementos visuais: fotografias de paisagem, horizontes, movimento
Eventos típicos: viagem, outdoor, lifestyle, escassez de tempo livre

### Arquétipo Amante
Energia: desejo, beleza, prazer, sofisticação
Cores dominantes: rosa, vinho, dourado rosé, off-white cremoso
Tipografia: serifa elegante + script para acentos
Elementos visuais: texturas sedosas, fotografias com luz quente, close-ups
Eventos típicos: moda, beleza, gastronomia, casamento, luxo acessível

## Paletas de Referência por Arquétipo

### Sistema de 6 Cores (obrigatório para todos)

Cada evento terá exatamente 6 cores definidas:

```
1. Âncora      — cor dominante, identidade (40% de uso)
2. Suporte     — complementa a âncora (25% de uso)
3. Ação        — botões, CTAs, destaques urgentes (10% de uso)
4. Fundo Princ — base das páginas e materiais (background)
5. Fundo Sec   — variação do fundo, seções alternadas
6. Texto       — tipografia principal
```

## Regras de Aplicação Visual

### LP do Evento
- Hero: fundo âncora ou fundo principal, headline em texto
- CTA botão: cor de ação com contraste mínimo 4.5:1 (WCAG AA)
- Formulário: fundo secundário ou branco
- Seções alternadas: fundo principal ↔ fundo secundário

### Emails
- Cor de destaque para nomes e datas: âncora ou ação
- Botão de CTA: cor de ação
- Fundo do email: branco ou fundo secundário (não fundo escuro)

### WhatsApp
- Sem formatação visual (é texto nativo)
- Nomes de produtos em *negrito*
- Horários em *negrito*

### Criativos Meta Ads
- Cor dominante: âncora (para reconhecimento de marca)
- Texto: máx 20% da área da imagem
- CTA no criativo: cor de ação
- Formato 1:1 (1080x1080) para feed + 9:16 (1080x1920) para stories

## Conceito Nomeado — Como Criar

O conceito visual do evento recebe um nome que captura a essência:
- 1-2 palavras em português
- Evoca emoção + posicionamento
- Serve como briefing verbal para qualquer designer

Exemplos:
- "Chama Seca" — energia, foco, sem adornos (Herói)
- "Tela em Branco" — possibilidade, clareza, intelectual (Sábio)
- "Noite Dourada" — sofisticação, mistério, transformação (Mago)
- "Campo Aberto" — expansão, natural, acolhedor (Explorador)
- "Pétala" — beleza, suavidade, feminino premium (Amante)
